Detailed Description

The Passenger class represents a passenger in the KYI Integrated system. It includes details such as the passenger's ID, type of ID, number of ID, full name, age type, and seat number.

This class provides constructors to create a Passenger object and getter and setter methods to access and modify the passenger's attributes.
